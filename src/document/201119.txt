Cron 초 분 시 일 월 요일
모든 값 : *
1 * * * * * : 매 1초 마다 실행( 1분 1초, 2분 1초, 3분 1초....)
*/5 * * * * * : 5초마다 실행


발효유 배치
cycle 정보를 조회하여 해당 애음 요일에 해당하는 일자를 계산
daily 테이블에 데이터를 입력

spring batch의 골격
reader  	: cycle테이블 정보를 조회
	SELECT *
	FROM cycle
processor  	: 애음요일을 실제 일자로 바꾸는 작업 ( 1==> 4~6)
writer  	: daily 테이블에 정보를 저장
