주제 : Isolation Level : 트랜잭션 격리 수준
--------------------------------------------------------------------------------------------------------------------------------------------------
트랜잭션이 다른 트랜잭션에게 어떻게 영향을 미치는지

정의: Isolation Level (고립화 레벨)

Isolation Level			Dirty Read		Nonrepeatable Read		Phantom Read
----------------------------------------------------------------------------
레벨0 Read Uncommited	발생				발생						발생
레벨1 Read Committed		X				발생						발생
레벨2 Repeatable Read	X				X						발생
레벨3 Serializable		X				X						X


트랜잭션 시작 : 월별부문, 본부, 팀, 파트별 실적 요약 레포트를
--------------------------------------------------
--월별부문에 대한 요약

1부문(300) - 1본부(100), 2본부(200)
SELECT 월,부문코드,Sum(판매금액)
FROM 실적
WHERE 월 = '202011'
 AND 조직 코드 IN( SELECT 조직코드
 				 FROM 조직
 				 WHERE 부문코드 = '영업1부문')
GROUP BY 월,부문코드;

--영업1부분에는 영업1본부,2본부가 존재
SELECT 월,본부코드,Sum(판매금액)
FROM 실적
WHERE 월 = '202011'
 AND 조직 코드 IN( SELECT 조직코드
 				 FROM 조직
 				 WHERE 부문코드 = '영업1본부')
GROUP BY 월,조직코드;
